# 🚀 AirChat-Go

基于 UDP 广播协议的局域网即时通讯工具，内置权限管理系统与文件共享引擎。

## ✨ 功能特性
- **零配置聊天**：基于 UDP 广播，无需服务器，局域网内自动发现。
- **权限管理 (OP)**：支持 `/op`、`/deop`、`/ban`、`/unban` 等指令。
- **文件共享**：一键开启 HTTP 文件服务器（`/send`），支持全员下载。
- **安全过滤**：内置 IP 封禁名单与并发锁，保护聊天环境。
- **彩色控制台**：适配 ANSI 转义码，交互体验极佳。

## 🛠️ 快速开始
1. 编译并运行：
   ```bash
   go build -o chat.exe chat.go
   ./chat.exe

    ```

2. 输入你的用户名（初始管理员名为 `air`）。
3. 使用 `/send` 分享当前目录文件。

## ⌨️ 管理员指令

* `/ban [IP]` - 封禁指定 IP
* `/op [User]` - 提升用户为管理员
* `/send` - 开启文件分享服务
